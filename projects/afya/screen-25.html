Privacidade e LGPD (Lei Geral de Proteção de Dados)

25/43

Direito dos titulares

Somente o titular pode fornecer suas informações às empresas para o consumo de produtos e serviços, porém, com a LGPD, ele terá clareza do uso dos dados fornecidos, tendo como direito:


Anonimização

Informações são convertidas em dados não identificáveis. Este processo é irreversível, desvinculando o dado da pessoa.

Eliminação

Informações são eliminadas completamente. Também um processo irreversível. O processo só não é possível quando há uma obrigação legal para mantê-lo. Nestes casos, a empresa pode optar por anonimizá-lo.

Portabilidade

Informações são movidas para outra empresa/serviço. Este processo pode ser reversível caso o titular opte por cancelar a portabilidade.

Automatização

Revisão de decisões automatizadas é quando um sistema possui regras de aceitação ou exclusão pré-definidas sem que um humano avalie.

Consentimento

Revogação do consentimento é retirar a permissão de tratamento de dados concedida anteriormente, como pedir a remoção de uma lista de e-mails que foi concedida pelo titular anteriormente.

Compartilhamento

Informação das entidades com as quais o controlador realizou o uso compartilhado dos dados.

Oposição

Oposição ao tratamento é optar por que seus dados não sejam tratados pela empresa antes mesmo que ela faça a coleta.

Confirmação

Confirmação da existência de tratamento é a possibilidade do titular questionar se a empresa possui dados dele ou não, sendo ela obrigada a informar verdadeiramente.

Correção

Correção dos dados permite que o titular mantenha seus dados atualizados e/ou corrija-os em caso de informação inexata.

Acesso

Acesso aos dados é poder visualizar e ter em mãos todos os dados que a empresa possui sobre você, como uma extração de dados pessoais.
